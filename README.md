# LifeStyling

A personal growth app combining mood tracking (nature elements), reading log, and habit building. Built with Next.js + Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Reflect** — Nature-element mood tracker with journal
- **Books** — Reading log with personal connection tracking
- **Habits** — Identity-based micro-habits with streak tracking
- **Harmony** — Gratitude garden and emotional connections
